# Naming Conventions

* **Language**: All classes, methods, properties, interfaces, packages, and other code elements must be written in **English**.
* **Interfaces**: Use the prefix `I` for all interface names (e.g., `IHeart`, `IDigestiveSystem`).
* **Domain Entities**: The names of classes and interfaces representing biological entities (e.g., proteins, enzymes, cells) must follow current scientific and biochemical naming conventions (e.g., using standard internationally recognized names or standard abbreviations).
